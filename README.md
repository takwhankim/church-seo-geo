# 교회 SEO/GEO 진단 스킬

교회 웹사이트의 SEO, AEO, GEO, 네이버 검색 최적화를 종합 진단하는 Claude AI 스킬입니다.

## 설치 방법

### Claude.ai
1. [church-seo-geo.skill 다운로드](링크)
2. Claude.ai → 사용자 지정 → 스킬 → 파일 업로드

### Claude Code
git clone https://github.com/본인계정명/church-seo-geo.git ~/.claude/skills/church-seo-geo

## 사용법
새 대화에서 아래와 같이 입력하세요:
- "이 교회 홈페이지 SEO 진단해 줘: https://교회URL"
- "강남역 교회 키워드로 검색 최적화 진단해 줘"

## 진단 영역
- 기본 SEO (메타태그, 구조화 데이터, 모바일 최적화)
- AEO (AI 답변 최적화)
- GEO (생성형 검색 최적화)
- 네이버 검색 최적화

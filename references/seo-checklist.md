# 기본 SEO 진단 체크리스트

교회 웹사이트의 기본 검색엔진 최적화 상태를 점검하는 항목이다.

## 1. 메타태그 (가중치: 높음)

### title 태그
- [ ] 존재 여부 확인
- [ ] 교회명 포함 여부
- [ ] 지역명 포함 여부 (예: "강남", "분당")
- [ ] 적정 길이: 30~60자 (한글 기준 15~30자)
- [ ] 각 페이지마다 고유한 title 사용 여부

### meta description
- [ ] 존재 여부 확인
- [ ] 교회 핵심 정보 포함 (위치, 예배시간, 교단)
- [ ] 적정 길이: 70~155자 (한글 기준 35~80자)
- [ ] 각 페이지마다 고유한 description 사용 여부

### Open Graph (og:) 태그
- [ ] og:title 존재 여부
- [ ] og:description 존재 여부
- [ ] og:image 존재 여부 (카카오톡 공유 시 썸네일)
- [ ] og:url 존재 여부

## 2. 구조화 데이터 (가중치: 높음)

### Church/Organization 스키마
- [ ] JSON-LD 형식의 구조화 데이터 존재 여부
- [ ] @type: Church 또는 Organization 사용 여부
- [ ] name, address, telephone 필드 포함 여부
- [ ] openingHours (예배시간) 포함 여부
- [ ] geo (위도/경도) 포함 여부
- [ ] sameAs (SNS 링크) 포함 여부

### 추가 스키마
- [ ] Event 스키마 (특별행사, 부흥회 등)
- [ ] Article/BlogPosting 스키마 (교회소식, 칼럼)
- [ ] BreadcrumbList 스키마

## 3. 기술 SEO (가중치: 중간)

### URL 구조
- [ ] 깔끔하고 의미 있는 URL 사용 여부 (예: /worship-info vs /page?id=123)
- [ ] 한글 URL 또는 영문 URL 일관성
- [ ] HTTPS 적용 여부

### 사이트맵과 robots.txt
- [ ] sitemap.xml 존재 여부 (/sitemap.xml 확인)
- [ ] robots.txt 존재 여부 (/robots.txt 확인)
- [ ] robots.txt에서 중요 페이지 차단 여부 점검

### 모바일 최적화
- [ ] viewport 메타태그 존재 여부
- [ ] 반응형 디자인 여부 (CSS media query 또는 별도 모바일 페이지)

### 페이지 속도 (간접 추정)
- [ ] HTML 파일 크기 (500KB 이하 권장)
- [ ] 외부 CSS/JS 파일 수 (과다 여부)
- [ ] 이미지 최적화 여부 (alt 태그 존재, 과도한 크기 추정)

## 4. 콘텐츠 SEO (가중치: 중간)

### 헤딩 구조
- [ ] h1 태그 존재 여부 (페이지당 1개 권장)
- [ ] h1에 교회명 또는 페이지 핵심 키워드 포함
- [ ] h2~h4 계층 구조 적절성

### 핵심 콘텐츠 페이지
- [ ] 교회 소개 페이지 존재 및 충실도
- [ ] 예배 안내 페이지 (시간, 장소, 주차 정보)
- [ ] 오시는 길 페이지 (지도 임베드, 주소, 대중교통)
- [ ] 교회소식/블로그 (정기 업데이트 여부)
- [ ] 연락처 페이지

### 이미지 최적화
- [ ] alt 속성 사용 여부
- [ ] alt 속성에 의미 있는 설명 포함 여부

## 5. 링크와 접근성 (가중치: 낮음)

### 내부 링크
- [ ] 주요 페이지 간 내부 링크 연결 여부
- [ ] 네비게이션 메뉴의 명확성

### canonical 태그
- [ ] canonical URL 지정 여부
- [ ] www/non-www 통일 여부

### 접근성 기본
- [ ] lang 속성 지정 여부 (html lang="ko")
- [ ] 이미지에 alt 텍스트 존재 여부 (SEO와 중복, 접근성 관점)

## 점수 산정

| 가중치 | 미충족 시 감점 | 해당 영역 |
|--------|--------------|-----------|
| 높음 | -15점 | 메타태그, 구조화 데이터 |
| 중간 | -8점 | 기술 SEO, 콘텐츠 SEO |
| 낮음 | -3점 | 링크, 접근성 |

100점에서 시작하여 미충족 항목만큼 감점한다. 최저 0점.

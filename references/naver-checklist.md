# 네이버 검색 최적화 진단 체크리스트

한국 교회의 온라인 가시성에서 네이버 검색은 구글만큼 중요하다. 네이버 검색 생태계 특성에 맞춘 최적화 항목을 점검한다.

## 1. 네이버 서치어드바이저 (가중치: 높음)

네이버 서치어드바이저(searchadvisor.naver.com)는 네이버 검색에 웹사이트를 등록하고 관리하는 도구다.

- [ ] 네이버 서치어드바이저에 사이트 등록 여부
- [ ] 사이트 소유 확인 완료 여부
- [ ] 사이트맵 제출 여부
- [ ] RSS 피드 제출 여부 (교회 소식/블로그 콘텐츠 색인용)
- [ ] 수집 요청 (웹페이지 수집 요청 기능 활용 여부)

## 2. 네이버 스마트플레이스 (가중치: 높음)

네이버 지도와 검색에 교회 정보를 노출하는 핵심 채널이다.

- [ ] 스마트플레이스 등록 여부
- [ ] 기본 정보 정확성 (교회명, 주소, 전화번호, 영업시간→예배시간)
- [ ] 카테고리 설정 적절성 ("종교시설 > 교회/성당" 등)
- [ ] 교회 사진 등록 여부 (외관, 내부, 주차장 등)
- [ ] 교회 소개 텍스트 작성 여부
- [ ] 예배 시간 정보 정확성
- [ ] 편의시설 정보 (주차, 엘리베이터, 수유실 등)
- [ ] 방문자 리뷰 관리 여부

## 3. 네이버 블로그 연동 (가중치: 높음)

네이버 검색에서 블로그 탭은 높은 노출 비중을 차지한다. 교회 공식 블로그 운영은 네이버 검색 가시성의 핵심이다.

- [ ] 교회 공식 네이버 블로그 존재 여부
- [ ] 블로그 포스팅 주기 (주 1회 이상 권장)
- [ ] 최근 포스팅 날짜 (3개월 이내 업데이트 여부)
- [ ] 포스팅에 교회명, 지역명 등 핵심 키워드 포함 여부
- [ ] 포스팅 내 이미지와 텍스트의 균형
- [ ] 블로그에서 공식 웹사이트로의 링크 존재 여부

## 4. 네이버 검색 콘텐츠 전략 (가중치: 중간)

네이버 검색 결과는 탭별(통합, 블로그, 카페, 지식인, 뉴스 등)로 나뉜다. 여러 탭에 노출되는 전략이 효과적이다.

### 네이버 카페
- [ ] 교회 관련 네이버 카페 운영 또는 참여 여부
- [ ] 지역 커뮤니티 카페에서의 교회 언급 여부

### 네이버 지식인 (참고용)
- [ ] 교회 관련 질문에 대한 공식 답변 활동 여부
  (예: "OO동 근처 교회 추천", "주일학교 있는 교회")

### 네이버 뉴스/프레스
- [ ] 교회 관련 보도자료나 뉴스 기사 존재 여부

## 5. 네이버 SEO 기술 요소 (가중치: 중간)

네이버 검색봇이 웹사이트를 올바르게 수집하기 위한 기술 요소다.

- [ ] 네이버 검색봇(Yeti) 접근 허용 여부 (robots.txt 확인)
- [ ] Open Graph 태그 설정 (네이버는 og 태그를 적극 활용한다)
- [ ] 네이버 신디케이션 핑 설정 여부 (콘텐츠 발행 시 자동 알림)
- [ ] HTML 기반 콘텐츠 여부 (플래시, iframe 과다 사용 지양)

## 6. 카카오 생태계 연동 (가중치: 낮음)

카카오톡/카카오맵은 네이버와 별도지만, 한국 사용자의 정보 공유 경로에서 중요하다.

- [ ] 카카오맵 장소 등록 여부
- [ ] 카카오톡 공유 시 썸네일(og:image) 정상 노출 여부
- [ ] 카카오톡 채널(플러스친구) 운영 여부

100점에서 시작하여 미충족 항목만큼 감점한다. 최저 0점.

## 네이버 진단의 한계

네이버 검색 결과 페이지는 web_search로 직접 조회할 수 없다. 따라서 네이버 영역의 진단은 다음 방식으로 수행한다:

1. **웹사이트 HTML 분석**: 네이버 관련 메타태그, robots.txt 내 Yeti 설정 등을 직접 확인
2. **구조적 권고**: 스마트플레이스, 블로그, 서치어드바이저 등록 여부는 사용자에게 확인을 요청
3. **ListeningMind 데이터**: MCP가 연결된 경우, 네이버 기반 검색 데이터를 활용해 키워드 전략 수립
4. **간접 확인**: web_search로 "site:blog.naver.com [교회명]" 등을 검색해 네이버 블로그 존재를 간접 확인

사용자에게 직접 확인이 필요한 항목은 보고서에 "확인 필요"로 명시하고, 체크 방법을 안내한다.

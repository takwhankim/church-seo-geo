# AEO/GEO 진단 체크리스트

AI 답변 최적화(AEO)와 생성형 검색 최적화(GEO) 관점에서 교회 웹사이트를 점검하는 항목이다.

## AEO (Answer Engine Optimization)

AI 기반 검색(구글 AI Overview, Bing Copilot 등)에서 교회 정보가 답변 소스로 선택될 가능성을 높이는 요소를 점검한다.

### 1. 질문-답변 구조 (가중치: 높음)

- [ ] FAQ 페이지 존재 여부
- [ ] FAQ에 실제 검색자가 물을 법한 질문 포함 여부
  - 예: "예배 시간이 언제인가요?", "주차 가능한가요?", "어린이 예배가 있나요?"
- [ ] 질문-답변이 구조화 데이터(FAQPage 스키마)로 마크업되어 있는지
- [ ] 답변이 간결하고 직접적인지 (2~3문장 이내로 핵심 답변 제공)

### 2. 직접 답변 가능 콘텐츠 (가중치: 높음)

- [ ] 예배 시간이 텍스트로 명확히 표기되어 있는지 (이미지가 아닌 텍스트)
- [ ] 교회 위치/주소가 텍스트로 표기되어 있는지
- [ ] 담임목사 이름과 소개가 텍스트로 있는지
- [ ] 교회 설립연도, 교단 정보가 명시되어 있는지
- [ ] 연락처(전화번호, 이메일)가 텍스트로 있는지

### 3. 콘텐츠 신뢰도 신호 (가중치: 중간)

- [ ] 콘텐츠 작성자/출처 표시 여부
- [ ] 최근 업데이트 일자 표시 여부
- [ ] 외부 권위 있는 사이트에서의 인용/링크 여부 (교단 공식 사이트 등)

## GEO (Generative Engine Optimization)

ChatGPT, Claude, Gemini 등 생성형 AI가 교회에 대한 질문을 받았을 때 정확한 정보를 제공할 수 있도록 하는 요소를 점검한다.

### 4. 인용 가능성 (가중치: 높음)

- [ ] 교회의 핵심 정보가 한 페이지에 정리되어 있는지
- [ ] 정보가 구조화되어 있는지 (제목-본문 계층, 리스트, 테이블)
- [ ] 정보에 모호함이 없는지 (주소가 정확한지, 시간이 구체적인지)
- [ ] 정보가 최신 상태인지 (작년 행사 일정이 그대로 남아 있지 않은지)

### 5. E-E-A-T 신호 (가중치: 중간)

E-E-A-T: 경험(Experience), 전문성(Expertise), 권위(Authoritativeness), 신뢰(Trustworthiness)

- [ ] 목사/사역자 프로필 페이지 존재 여부 (학력, 경력)
- [ ] 교회 역사/연혁 페이지 존재 여부
- [ ] 교단 소속 명시 여부
- [ ] HTTPS 사용 여부
- [ ] 개인정보처리방침 페이지 존재 여부

### 6. 멀티모달 콘텐츠 (가중치: 중간)

- [ ] 설교 텍스트(전문 또는 요약)가 웹에 공개되어 있는지
- [ ] 설교 영상에 제목, 설명, 태그가 적절히 있는지
- [ ] 유튜브 채널 연동 및 설명란 최적화 여부
- [ ] 설교 음성(팟캐스트)에 쇼노트/요약 텍스트가 있는지

### 7. 크로스 플랫폼 일관성 (가중치: 낮음)

- [ ] 구글 비즈니스 프로필(Google Business Profile) 등록 및 정보 일치 여부
- [ ] 네이버 스마트플레이스 정보와 웹사이트 정보 일치 여부
- [ ] SNS 프로필(인스타그램, 유튜브, 페이스북)의 교회 정보 일치 여부
- [ ] 위키백과/나무위키 등 지식 베이스에 교회 정보 존재 여부

## 점수 산정

| 가중치 | 미충족 시 감점 | 해당 항목 |
|--------|--------------|-----------|
| 높음 | -15점 | 질문-답변 구조, 직접 답변 콘텐츠, 인용 가능성 |
| 중간 | -8점 | 신뢰도 신호, E-E-A-T, 멀티모달 |
| 낮음 | -3점 | 크로스 플랫폼 일관성 |

100점에서 시작하여 미충족 항목만큼 감점한다. 최저 0점.

## 교회 특화 GEO 권고사항 (보고서 작성 시 참고)

교회 웹사이트가 AI에게 "좋은 출처"로 인식되려면 다음이 중요하다:

1. **검색자의 질문에 직접 대응하는 콘텐츠**: "강남역 근처 교회"를 검색하는 사람은 위치, 예배시간, 교통편을 원한다. 이 정보가 텍스트로 명확히 있어야 한다.

2. **정보의 최신성**: 2023년 부활절 행사 안내가 메인에 남아 있으면 AI는 이 사이트의 정보 신뢰도를 낮게 판단할 수 있다.

3. **정보의 구조화**: 예배 시간표를 이미지로만 올리면 AI가 읽을 수 없다. HTML 테이블이나 구조화 데이터로 제공해야 한다.
